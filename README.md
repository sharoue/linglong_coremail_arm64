# linglong_coremail_arm64
- 玲珑化的coremail（论客邮箱）客户端(打包参数)
- 在信创的uos 1070版本上，论客邮箱客户端因为GTK的bug问题，会导致界面出现黑块，而显然没人会去修debian 10时代的GTK的问题了，只能通过玲珑包来提供高版本GTK。
- 官方玲珑仓库有x86版，可以直接使用。但是没有arm64版，提供arm64版的打包参数，供大家自行打包。基于org.deepin.base//25.2.2.6。
- 如果目标的机器是华为W585X一类的SOC，还可能因为桌面锁定Wayland问题无法启动，需要给执行文件增加指定x11方式的补丁。
- 感谢AI。
- 目前可以查看邮件，可以另存附件，但是点击附件无法打开，应该是呼叫打开方式失败，暂时没有解决办法，只能当参考了。
